简介

Squid是Linux自带的代理软件，与其他代理软件如Apache、Socks等相比，下载安装简单，配置灵活，支持缓存和多种协议。

解决方法

启动squid服务时提示如下错误

/usr/sbin/squid: relocation error: /usr/sbin/squid: symbol SSL_set_alpn_protos, version libssl.so.10 not defined in file libss…e reference

原因：未安装openssl

解决方法：yum install openssl

安装后重新启动squid即可：systemctl start squid

Windows端

查询下未设置代理前的ip地址

在Chrome浏览器中设置代理，点击右上角三个点–>设置，滑到最底点击高级，找到打开代理设置–>局域网设置，勾选上为LAN使用代理服务器，下面填写代理服务器的地址和端口即可。

保存后再查询ip地址，已经发生了改变，说明成功设置代理！
